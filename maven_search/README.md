# maven_search

## 介绍
用go编写的CLI 用于查找maven仓库包, 具有TUI 


# 快速开始

-  [下载安装包使用](https://gitee.com/SuperWindcloud/maven_search/releases/download/1.0.00/maven_search.exe)


-  Scoop  安装使用 

    ```
    scoop bucket add hy https://gitee.com/SuperWindcloud/hyperscoop_bucket.git
    scoop install maven_search
    ```
